# BLIP Vision-Language Application

## Description
A complete vision-language application using BLIP (Bootstrapping Language-Image Pre-training) for image captioning and visual question answering.
## Features
- Image upload (local or URL)
- Automatic image caption generation
- Visual Question Answering
- Interactive menu system
- Sample questions and custom questions

## How It Works
1. User uploads an image (local file or URL)
2. BLIP model generates a caption
3. User can ask questions about the image
4. Model answers based on visual content
